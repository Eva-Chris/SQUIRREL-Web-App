---VERSIONS---
Python Version: 3.8.10
NumPy Version: 1.19.5
Tensorforce Version: 0.6.5
gym Version: 0.26.0
Ubuntu Version: 20.04.5
IDE: Visual Studio Code
Flask Version: 2.2.2

---ACTIVATING VIRTUAL ENVIRONMENT---
If env folder is not available type: python3 -m venv env

To activate the virtual environment type: source env/bin/activate

If flask is not installed type: pip install flask-sqlalchemy
If flask-wtf is not installed type: pip install flask-wtf
If flask-mail is not installed type: pip install flask-mail

---CHANGING ENVIRONMENT---
In Visual Studio Code press Ctrl+Shift+P and select venv as Python Intepreter

---OPENING THE WEBSITE---
Type: python3 app/app.py

The server starts on: http://127.0.0.1:5000 or localhost:5000
